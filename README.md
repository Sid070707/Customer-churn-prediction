# Customer Churn Prediction

This project predicts whether a telecom customer is likely to churn using machine learning.

## Model
Logistic Regression

## Accuracy
79%

## Features Used
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Streaming Services

## Deployment
Streamlit web app for real-time predictions.
